# this-day-in-wallpaper

Daily AMOLED wallpaper for Android, generated by Gemini AI.  
Each morning a new 1080×2340 wallpaper is created based on the most interesting science, medicine, or technology fact of the day — visualised as a glowing neon infographic on a pure black background.

## How it works

1. **GitHub Actions** runs at 7 AM UTC every day
2. `generate.py` asks Gemini to pick the day's most fascinating fact
3. Gemini Imagen 3 generates a portrait AMOLED wallpaper for that fact
4. The image is committed to this repo as `wallpaper/latest.png`
5. Tasker on Android fetches the stable raw URL and sets it as the wallpaper

## Setup

### 1. Add your Gemini API key

In your GitHub repo: **Settings → Secrets and variables → Actions → New repository secret**

| Name | Value |
|------|-------|
| `GEMINI_API_KEY` | your Google AI Studio key |

### 2. Test locally

```bash
pip install -r requirements.txt
export GEMINI_API_KEY=your_key_here
python generate.py
```

### 3. Trigger manually

Go to **Actions → Generate Daily Wallpaper → Run workflow** to generate your first image without waiting for the cron.

### 4. Tasker setup (Android)

Create a Tasker profile that runs daily (e.g. at 7:30 AM) with these actions:

1. **HTTP Request** (or *HTTP Get*)
   - URL: `https://raw.githubusercontent.com/fpwilson3-star/this-day-in-wallpaper/main/wallpaper/latest.png`
   - Output file: `/sdcard/Pictures/wallpaper_today.png`
2. **Set Wallpaper**
   - Image: `/sdcard/Pictures/wallpaper_today.png`

## Output

| File | Description |
|------|-------------|
| `wallpaper/latest.png` | Always today's wallpaper (stable URL for Tasker) |
| `wallpaper/YYYY-MM-DD.png` | Dated archive |
| `wallpaper/latest_fact.txt` | Date + fact text |
